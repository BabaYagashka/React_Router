# Ghost of Tsushima React Landing Page

A responsive React-based landing page inspired by the action-adventure game *Ghost of Tsushima*. This project showcases the game, its protagonist **Jin Sakai**, and provides sections like About, Contact, and GitHub integration. Built using **React** and **Tailwind CSS** for a modern, mobile-friendly design.

---

## Features

- Hero section with **Download Now** button linking to the official game site  
- **About Jin Sakai** section detailing the game story  
- **Contact form** for visitors to get in touch  
- **GitHub integration** to show follower count and profile picture  
- Fully **responsive design**  
- Clean UI using **Tailwind CSS**  

---

## Technologies Used

- React  
- Tailwind CSS  
- React Router DOM  
- GitHub API
